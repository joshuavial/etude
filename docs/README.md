# etude docs

This directory holds user-facing documentation for implemented behavior.

## Sections

- [Plans](plans/README.md) - notes on planned components that do not exist yet.

There are no additional user-facing guides yet because the implemented
user-facing CLI is still limited to root help and version output. The current
implemented state is summarized in the top-level [README](../README.md).

The storage and manifest packages that exist today are Go APIs internal to this
module. The top-level README mentions them only as implementation status; there
are no user-facing command docs for them.
