Open-source software changed the economics of the industry.
When source code is freely available, every developer can inspect, adapt, and improve it.
Bugs get found faster because more eyes are looking.
Features get built because someone somewhere needed them and had the freedom to contribute.
The shared infrastructure of the internet runs largely on open-source foundations.
Licenses define the terms of sharing, from permissive allowance of commercial use to copyleft requirements that keep derivatives open.
Understanding the license is as important as understanding the code.
